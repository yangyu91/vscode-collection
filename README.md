# VS Code 离线安装包与插件合集 (Windows 7 & Windows 10/11)

所有文件已上传至本仓库的 **Releases** 页面，并按系统版本进行了重命名。

## 下载地址

👉 **[点击前往下载页面 (Releases v1.1.0)](https://github.com/yangyu91/vscode-collection/releases/tag/v1.1.0)**

## 包含内容

### 1. VS Code 安装程序
- **VSCode_Win7_v1.70.2.exe**: 适配 Windows 7
- **VSCode_Win10_11_Latest.exe**: 适配 Windows 10/11

### 2. 插件列表 (VSIX)
每个插件都提供了两个版本，请根据您的系统选择：
- **Intelephense**: PHP 智能感知
- **HTML CSS Support**: HTML/CSS 支持
- **JavaScript (ES6) Snippets**: JS 代码片段
- **SQLTools**: 数据库管理工具
- **Chinese Language Pack**: 简体中文语言包

## 使用说明
1. 下载对应系统的安装包并安装。
2. 下载文件名中带有您系统标识（Win7 或 Win10_11）的 `.vsix` 插件文件。
3. 在 VS Code 中，点击插件面板右上角的 "..." -> "Install from VSIX..." 进行安装。
