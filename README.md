# VS Code 离线安装包与插件合集 (Windows 7 & Windows 10/11)

由于安装包体积较大，所有文件已上传至本仓库的 **Releases** 页面。

## 下载地址

👉 **[点击前往下载页面 (Releases)](https://github.com/yangyu91/vscode-collection/releases/tag/v1.0.0)**

## 包含内容

### 1. VS Code 安装程序
- **Windows 7**: 官方最后支持版本 (1.70.2)
- **Windows 10/11**: 最新稳定版

### 2. 插件列表 (VSIX)
仓库已为不同系统匹配了兼容的插件版本：
- **Intelephense**: PHP 智能感知
- **HTML CSS Support**: HTML/CSS 支持
- **Path Intellisense**: 路径自动补全 (Win7 暂缺，Win10 已包含)
- **JavaScript (ES6) Snippets**: JS 代码片段
- **SQLTools**: 数据库管理工具
- **Chinese (Simplified)**: 简体中文语言包

## 使用说明
1. 下载对应系统的安装包并安装。
2. 下载所需的 `.vsix` 插件文件。
3. 在 VS Code 中，点击插件面板右上角的 "..." -> "Install from VSIX..." 进行安装。
